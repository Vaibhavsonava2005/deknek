# DekNek

> Next-Gen Digital Solutions — Built with 3D animations, real-time auth, and modern web technologies.

## Developed By
👨‍💻 **Vaibhav** (Vaibhavsonava2005)

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **3D & Animation**: Three.js, React Three Fiber, Framer Motion
- **Auth & Database**: Supabase
- **Styling**: Vanilla CSS with glassmorphism design system
- **Deployment**: Vercel

## Features

- 🎨 Stunning 3D animated hero with floating geometry & particles
- 🔐 Real user authentication (signup, login, protected dashboard)
- 📊 Interactive dashboard with user profiles
- 📬 Contact form with database storage
- 📱 Fully responsive design
- ✨ Scroll-driven animations & glassmorphism UI

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Create `.env.local` with:

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```
